Arrow Functions

This arrow function replaces all the spaces in a string with dashes instead.

Named Function

            function replaceSpaces(firstString){
                return firstString.replaceAll(" ", "-");
            }

Function Expression

            const replaceSpaces = function(firstString){
                return firstString.replaceAll(" ", "-");
            }

Arrow Function

            const replaceSpaces = firstString => firstString.replaceAll(" ", "-");
